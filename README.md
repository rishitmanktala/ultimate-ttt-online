# 🎮 Ultimate Tic Tac Toe — Online Multiplayer

**Play Ultimate Tic Tac Toe against friends in real-time** with Socket.io multiplayer, confetti celebrations, and a sleek dark UI.

This is the **client-side** app for the online multiplayer version. It connects to the [ultimate-ttt](https://github.com/rishitmanktala/ultimate-ttt) Socket.io server.

---

## ✨ Features

- 🌐 **Real-Time Multiplayer** — Play against friends via Socket.io rooms
- 🏆 **Win Detection** — Automatic win/draw detection with confetti animations
- 🎨 **Premium Dark UI** — Space Grotesk font, glowing accents, glassmorphism panels
- 📱 **Responsive Design** — Adapts seamlessly to mobile and desktop
- ⚡ **Tailwind CSS** — Utility-first styling via CDN

---

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/rishitmanktala/ultimate-ttt-online.git
   ```

2. **Start the multiplayer server** (see [ultimate-ttt](https://github.com/rishitmanktala/ultimate-ttt)):
   ```bash
   git clone https://github.com/rishitmanktala/ultimate-ttt.git
   cd ultimate-ttt
   npm install
   npm start
   ```

3. Open `index.html` in a browser and connect to your server.
